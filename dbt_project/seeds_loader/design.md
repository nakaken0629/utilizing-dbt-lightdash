# ローダーの仕様

ローダーは、データベースdbに登録された情報を、データウェアハウスdb_dwhのrawスキーマに読み込むツールです。

## ツールの仕様

- 言語はPythonを利用する。
- パッケージマネージャとしてuvを利用する

## 初期化ツール

- db_dwhにpublic_rawスキーマを作成する。すでにある場合は中にある全てのテーブルの削除を事前に行う

## データ投入ツール

dbにある以下のテーブルの内容を、そのままコピーします。

- 会員テーブル
- 会員ステータス変更履歴テーブル
- カテゴリテーブル
- 食品テーブル
- 購入テーブル
- 購入明細テーブル
